# Advanced

Neon OSS Kit is a collection of tools and services that help you build and scale your applications with Neon. And this tools can be hechful for you to build and scale your applications with Neon if no proper documentation is provided.

This section provides an overview of the tools and services available in the Neon OSS Kit, as well as some advanced topics to help you get the most out of Neon OSS Kit.

## Concepts and Features

Neon OSS Kit offers a range of tools and services to help you build and scale your applications. Some of the key concepts and features include:

| Tool/Service | Description | Documentation |
|--------------|-------------|---------------|
| [Neon](/neon) | Neon is a serverless Postgres platform designed to help you build reliable and scalable applications faster. We separate compute and storage to offer modern developer features such as autoscaling, branching, point-in-time restore, and more. You can get started with Neon by creating a free account on the [official website](https://console.neon.tech). | [Docs](https://neon.tech/docs/guides/neon-features) |
| [Next.js](/nextjs) | Next.js is a React framework that allows you to build static and dynamic web applications with ease. It offers features such as server-side rendering, static site generation, and API routes to help you build fast and scalable web applications. You can get started with Next.js by following the [official documentation](https://nextjs.org/docs). | [Docs](https://nextjs.org/docs) |
| [Prisma ORM](/prisma) | Prisma is an open-source ORM for Node.js and TypeScript that allows you to interact with your database using a type-safe API. It offers features such as schema migrations, query builders, and data validation to help you build scalable and maintainable applications. You can get started with Prisma by following the [official documentation](https://www.prisma.io/docs). | [Docs](https://www.prisma.io/docs) |
| [Drizzle ORM](/drizzle) | Drizzle is a lightweight ORM for Node.js and TypeScript that allows you to interact with your database using a simple API. It offers features such as schema migrations, query builders, and data validation to help you build fast and scalable applications. You can get started with Drizzle by following the [official documentation](https://orm.drizzle.team/docs/overview). | [Docs](https://orm.drizzle.team/docs/overview) |
| [Clerk Auth](/clerk) | Clerk is an authentication and user management service that allows you to add authentication to your applications with ease. It offers features such as social login, email verification, and role-based access control to help you build secure and scalable applications. You can get started with Clerk by following the [official documentation](https://clerk.dev/docs). | [Docs](https://clerk.dev/docs) |
| [Tailwind CSS](/tailwindcss) | Tailwind CSS is a utility-first CSS framework that allows you to build custom designs without writing any CSS. It offers features such as utility classes, responsive design, and dark mode to help you build beautiful and responsive web applications. You can get started with Tailwind CSS by following the [official documentation](https://tailwindcss.com/docs). | [Docs](https://tailwindcss.com/docs) |
| [ShadcnUI](/shadcnui) | ShadcnUI is a React component library that allows you to build beautiful and responsive user interfaces with ease. It offers features such as customizable themes, dark mode, and internationalization to help you build accessible and inclusive web applications. You can get started with ShadcnUI by following the [official documentation](https://ui.shadcn.com/docs). | [Docs](https://ui.shadcn.com/docs) |
