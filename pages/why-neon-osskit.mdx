# Why Neon OSS Starter Kit?

To understand why Neon OSS Starter Kit is a good choice for your next project, let's take a look at some of the features it offers.

## Features

- **Zero Configuration**: Neon OSS Starter Kit comes with a pre-configured setup that allows you to start building your project without any additional configuration.
- **Built-in Components**: It includes a set of built-in components that you can use to build your project.
- **Customizable**: You can easily customize the starter kit to fit your project requirements.
- **Easy to Use**: Neon OSS Starter Kit is easy to use and comes with detailed documentation to help you get started.

## The Stack Neon OSS relies on

Neon OSS Starter Kit is built on top of several technologies that help you build modern web applications faster and more efficiently.

- **React**: Neon OSS Starter Kit is built using React, a popular JavaScript library for building user interfaces.
- **Next.js**: It is built on top of Next.js, a React framework that provides features like server-side rendering, static site generation, and more.
- **Tailwind CSS**: Neon OSS Starter Kit uses Tailwind CSS for styling, a utility-first CSS framework that helps you build custom designs without writing any CSS.
- **Neon**: Neon is a serverless Postgres platform designed to help you build reliable and scalable applications faster. We separate compute and storage to offer modern developer features such as autoscaling, branching, point-in-time restore, and more.
- **ORMs**: Neon OSS Starter Kit uses Prisma and Drizzle as ORMs to interact with the database.
  - **Prisma**: Prisma is a modern database toolkit that makes database access easy with type-safe database access and declarative data modeling.
  - **Drizzle**: Drizzle is a lightweight ORM for Node.js that provides a simple and intuitive API for interacting with databases.
- **Authentication**: Neon OSS Starter Kit includes authentication using Clerk, a modern authentication platform that provides a secure and customizable authentication experience.
- **ShadcnUI**: Neon OSS Starter Kit uses ShadcnUI for building the user interface, a set of React components that help you build beautiful and responsive designs.
- **ESLint and Prettier**: Neon OSS Starter Kit includes ESLint and Prettier for code linting and formatting to ensure consistent code quality.

## Why Neon?

Neon is a serverless Postgres built for the cloud.

- **Modern Developer Features**: Neon offers modern developer features like autoscaling, branching, point-in-time restore, and more to help you build reliable and scalable applications faster.
- **Separate Compute and Storage**: Neon separates compute and storage to offer better performance and scalability for your applications.
- **Built-in Security**: Neon provides built-in security features like encryption at rest, network isolation, and more to keep your data secure.
- **Developer Experience**: Neon offers a great developer experience with features like a web-based console, CLI, and more to help you build and manage your applications easily.
- **Community Support**: Neon has a growing community of developers who are ready to help you with any questions or issues you may have.
